# Ship It Studio — agency site

A single-file, no-build website that showcases your work with live links and captures leads. Open `index.html` to view it. Deploys to Vercel in one click.

## Switch-ons (5 min, all in `index.html`)

1. **WhatsApp number** — find `const WHATSAPP = "447399479952"` and replace with your number (international format, no `+`).
2. **Contact form** — get a free key at https://web3forms.com (enter triggsoliver@gmail.com), then replace `REPLACE_WITH_YOUR_WEB3FORMS_ACCESS_KEY`.
3. **Project links** — edit the `PROJECTS` array; set each `url` to the live site. Blank = "coming soon"; `private:true` = hidden link.

## Deploy

Import this repo at https://vercel.com/new (Framework: Other) and Deploy. Add `shipitstudio.co.uk` under Settings → Domains.
